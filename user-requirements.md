# User Requirements – Movie/TV Show Tracker Web App

## 1. User Goals
- Easily search for movies and TV shows using a prominent search bar in the header.
- Maintain a personal watchlist of titles to watch with clear add/remove actions.
- Track progress by marking items as watched with intuitive buttons.
- View a summary of current and completed titles in a clean dashboard.
- Discover trending or popular movies/TV shows via a vertical slider on the homepage.
- Enjoy a clean, modern, and minimal UI that is easy to navigate.

---

## 2. User Stories
1. **As a user,** I want to search for movies and shows so that I can find information quickly.  
2. **As a user,** I want to add movies/shows to my watchlist so I can remember what I want to watch later.  
3. **As a user,** I want to move movies/shows from my watchlist to my watched list once I’ve seen them.  
4. **As a user,** I want to see a dashboard summarizing my progress so I can track what I’ve completed.  
5. **As a user,** I want to see trending titles automatically so I can discover new movies and shows.  
6. **As a user,** I want the app to be mobile-friendly so I can use it on my phone.  

7. **As a user,** I want the homepage to feature a header section with the site logo, a central search bar, and navigation menus for easy access.
8. **As a user,** I want to see trending movies and shows in a vertical slider for quick discovery.
9. **As a user,** I want the main content to display movies and TV shows in a clean grid layout with clear buttons to add to my watchlist or mark as watched.

---

## 3. Acceptance Criteria
- Users can perform searches and receive results from the external API within 2 seconds.  
- Users can add/remove items from watchlist and watched list.  
- Dashboard displays watchlist and watched count correctly.  
- Trending titles refresh automatically (daily or scheduled).  
- App works on desktop and mobile browsers.  

- Homepage features a clean header with logo, search bar, and navigation menus.
- Trending movies/shows are displayed in a vertical slider.
- Main content uses a grid layout for movie/TV show info with clear action buttons.
- UI is minimal, modern, and easy to use.

---

## 4. Assumptions
- Only one user account is required for MVP (multi-user login optional).  
- Internet connection required for API data.  
- API rate limits will not be exceeded for demonstration purposes.  

---

## 5. Constraints
- Must use publicly available APIs (TMDB/OMDb).  
- Free-tier cloud infrastructure for deployment.  
- Minimal authentication/authorization for MVP (can be expanded later).
